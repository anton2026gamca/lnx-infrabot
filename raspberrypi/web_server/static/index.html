This calibration calculates the camera focal length to accurately measure distance to the goal.

  1. Place the robot near your own goal (starting position)
  2. Set the initial distance value (distance from robot to enemy goal in mm)

  4. Drive the robot straight toward the enemy goal
  5. When line sensors detect the line, stop calibration
